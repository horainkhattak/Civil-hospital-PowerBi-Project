Civil Hospital Lahore Dashboard
Overview

The Civil Hospital Lahore Dashboard is an interactive healthcare analytics solution designed to monitor and analyze hospital operations. It provides visual insights into patient records, bed occupancy, diagnosis trends, doctor feedback, and financial performance, enabling efficient decision-making and resource management.

Features
Patient-wise data filtering using Patient ID.
Date range selection for customized analysis.
Admission, discharge, and follow-up tracking.
Bed occupancy monitoring across different ward categories.
Doctor-wise feedback analysis.
Diagnosis-wise patient distribution visualization.
Billing and health insurance amount comparison.
Interactive charts and KPI cards for quick insights.
Dashboard Components
Key Performance Indicators (KPIs)
Admit Date
Discharge Date
Follow-Up Date
Total Bill Amount
Analytical Visualizations
Bed Occupancy Analysis: Displays occupancy levels for Private, General, and ICU beds.
Doctor Feedback Distribution: Shows feedback volume received by individual doctors.
Diagnosis-wise Patient Count: Highlights the number of patients for each diagnosis category.
Financial Analysis: Compares Billing Amount and Health Insurance Amount across medical services such as CT Scan, Ultrasound, MRI, Blood Test, and X-Ray.
